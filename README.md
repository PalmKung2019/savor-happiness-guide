# Savor Happiness Guide
**In Minburi & Nongchok | Featuring 20 Cafés & Restaurants**

"ความสุข... บางครั้งไม่จำเป็นต้องเดินทางไปให้ไกลสุดตา เพียงแค่เราลองหยุดพักเพื่อ “ละเลียดชิม (Savor) บรรยากาศรอบข้างในจังหวะที่พอดี"

โปรเจกต์นี้คือส่วนหนึ่งของ Thesis สาขาการออกแบบกราฟิก คณะดิจิทัลมีเดีย มหาวิทยาลัยศรีปทุม ที่ถ่ายทอดเสน่ห์ของย่านชานเมืองผ่านมุมมองใหม่ ผสมผสานระหว่างงาน Graphic Design, Photography และ Web Development เข้าด้วยกัน

## 📖 About The Project
**Savor Happiness** คือบันทึกการเดินทางที่รวบรวม 20 พิกัดลับใน 2 ย่านชานเมืองกรุงเทพฯ เพื่อให้ทุกคนได้หลีกหนีความวุ่นวายมาพักใจ:
- 🏙️ **Minburi (มีนบุรี):** สัมผัสความลงตัวระหว่างความทันสมัยและกลิ่นอายดั้งเดิม ตั้งแต่คาเฟ่ดีไซน์เท่ไปจนถึงร้านอาหารริมน้ำ
- 🍃 **Nongchok (หนองจอก):** ค้นพบความสุขในจังหวะที่ช้าลง (Slow Life) ท่ามกลางทุ่งหญ้าสีเขียวและอากาศบริสุทธิ์

## 🎨 Deliverables
โปรเจกต์นี้ถูกออกแบบมาเพื่อสร้างประสบการณ์แบบครบวงจร (Visual Storytelling & Physical Experience):
1. **The Guidebook:** หนังสือคู่มือขนาด A5 จำนวน 126 หน้า พิมพ์ 4 สี พร้อมการจัดหน้าแบบ Dynamic Layout
2. **Merchandise:** โปสการ์ดแผนที่ย่านมีนบุรี-หนองจอก, ที่คั่นหนังสือ (Bookmark) และสติ๊กเกอร์
3. **Website Platform:** เว็บไซต์แนะนำข้อมูลร้านรองรับ Responsive Design (Mobile, Tablet, Desktop) 

🌐 **Live Website:** [https://savor-happiness-guide.vercel.app](https://savor-happiness-guide.vercel.app)
